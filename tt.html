Text Type Analysis Flowchart

Starting Point
Start: Analyze the text
Step 1: Check for Explicit Questions
Does the text contain explicit questions?
If YES: Go to Step 2 (Questions that are clearly directed at someone, possibly the reader, and may be marked with a ?) | If NO: Go to Step 5
Step 2: Assess Question Format Dominance
Are questions the main format?
If YES: Go to Step 3 (Most of the text is explicit questions) | If NO: Go to Step 11
Step 3: Check for Responses
Are responses included or expected?
If YES: → INTERROGATIVE
If NO: Go to Step 4
Step 4: Search/Query Check
Is it a search/prompt query?
If YES: → INTERROGATIVE
If NO: Go to Step 11
Step 5: How-to Format Check
Does it follow how-to/step-by-step format?
If YES: Go to Step 6 (A clear, methodical series of instructions that should be followed) | If NO: Go to Step 7
Step 6: Goal Orientation Check
Goal is final result/completion?
If YES: → INSTRUCTIONAL
If NO: Go to Step 11
Step 7: Story/Sequence Check
Does it tell a story or cohesive sequence of events?
If YES: Go to Step 8 | If NO: Go to Step 10
Step 8: Events vs. Descriptions Priority
Is it more concerned with events than single descriptions?
If YES: Go to Step 9 | If NO: Go to Step 11
Step 9: Fiction Check
Is it fictional or literary?
If YES: → NARRATIVE
If NO: → NARRATIVE
Step 10: Persuasion Check
Is it trying to convince or persuade?
If YES: Go to Step 11a (A point of view is evident, and the text either supports or opposes it) | If NO: Go to Step 13
Step 11a: Emotional Language Check (Persuasion Branch)
Uses emotional/emotive language?
If YES: → PERSUASIVE
If NO: Go to Step 12
Step 12: Logical Arguments Check
Uses logical arguments?
If YES: → ARGUMENTATIVE
If NO: → PERSUASIVE
Step 13: Information Purpose Check
Primary purpose is to inform?
If YES: Go to Step 14 (The text contains facts, or is presented as just giving facts) | If NO: Go to Step 11
Step 14: Context and Detail Check
Provides context and detailed information?
If YES: Go to Step 15 | If NO: Go to Step 16
Step 15: Fact-Based Check
Based on facts, not emotions?
If YES: → EXPOSITORY
If NO: Go to Step 11
Step 16: Specific Details Check
Gives specific details about object/person/situation?
If YES: Go to Step 17 | If NO: → EXPOSITORY
Step 17: Non-Emotive Language Check
No emotive language?
If YES: → DESCRIPTIVE
If NO: Go to Step 11
Step 11: Multiple Characteristics Review
Action: Review multiple characteristics and choose best fit based on dominant features
If no clear match: → OTHER

Text Type Categories

1. INTERROGATIVE
Question-based formats and inquiry-driven content
  • Interviews
  • Exams and quizzes
  • Q&A formats
  • Search queries
2. INSTRUCTIONAL
Goal-oriented content designed to teach or guide
  • How-to guides
  • Step-by-step processes
  • Tutorials
  • Manuals
3. NARRATIVE
Story-telling and event sequences
  • Stories and novels
  • Historical accounts
  • TV/film/literary works
  • Personal experiences
4. PERSUASIVE
Content designed to influence through emotion
  • Advertisements
  • Opinion pieces
  • Personal reviews
  • Political speeches
5. ARGUMENTATIVE
Logic-based content that presents structured arguments
  • Academic essays
  • Debates
  • Legal documents
  • Research papers
6. EXPOSITORY
Informational content that explains or informs
  • News articles
  • Encyclopedias
  • Textbooks
  • Reports
7. DESCRIPTIVE
Content that creates vivid images through detailed description
  • Character descriptions
  • Setting descriptions
  • Product specifications
  • Scientific observations
8. OTHER
Texts that do not fit clearly into any specific category
  • Mixed or hybrid text types
  • Experimental writing
  • Abstract expressions
  • Undefined content formats
