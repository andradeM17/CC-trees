Text Classification System

Starting Point
Rule 1: Is this digital content?
If Yes: Go to Rule 2 (Content that requires some form of technology to view)
If No: Go to Rule 4
Rule 2: Is this content exclusively online?
Content visible only on the web
If Yes: Go to Rule 3 (website homepage, forum post, search query)
If No: Go to Rule 5
Rule 3: Was this authored by an authority?
Operating on a more 'professional' scale than an individual person
If Yes: → WEBSITE (Go to Rule 100)
If No: Go to Rule 6 (social media post, forum post, social media subtitles)
Rule 4: Where would you expect to see this type of content?
Social Media → Rule 200 | Website → Rule 100 | News → Rule 300 | Marketing → Rule 400 | Review → Rule 500 | Legal → Rule 600 | Instructions → Rule 700 | Subtitles → Rule 800 | Literary → Rule 900 | Medical → Rule 1000 | Encyclopedia → Rule 1100 | Other → Rule 1200
Rule 5: What type of digital content is it?
Software → Digital - Software | Video Game → Digital - Video Game Interface | Archive → Digital - Archive | Subtitles → Rule 800
Rule 6: Does it contain some form of peer-to-peer communication?
Can be explicit/implicit, doesn't need visual confirmation
If Yes: → SOCIAL (Go to Rule 200)
If No: Go to Rule 7 (writer to reader communication)
Rule 7: Is this a collection of texts?
Can be implicit, not necessary for multiple texts in sample
If Yes: → Digital - Archive
If No: Go to Rule 4 (Singular text: recipe, short story, speech)

Social Media Classification (Rules 200-299)

Rule 200: Search results, feed or post?
Search results → Social Media - Search Results (Several short texts, unrelated but same topic)
Feed → Rule 201 (Several texts, unrelated, different topics)
Post → Rule 202 (One text)
Rule 201: Are posts from same user?
If Yes: → Social Media - Profile (Usernames, multiple posts continuing from each other)
If No: → Social Media - Feed
Rule 202: Someone asking question expecting answer?
If Yes: → Website - Forum
If No: → Social Media - Post

Website Classification (Rules 100-199)

Rule 100: What type of text is it?
Forum → Website - Forum (Question expecting answer on niche topic)
FAQ → Website - FAQ (Series of commonly asked questions and answers)
Blog → Website - Blog (Personal/professional, longer form, single author)
Search query → Website - Search Query (Short text, often question)
Products/services → Rule 101 | Instructional → Rule 102
Fiction → Website - Fiction
Creative Nonfiction → Website - Creative Nonfiction (Real events using literary techniques)
Listings → Website - Index (List of categories, not specific products)
Homepage → Website - Homepage (Landing page with general info and links)
Template → Website - Boilerplate (Text that doesn't differ between websites)
Other → Website - Article
Rule 101: Specific products with prices or categories?
Specific products → Rule 103
Categories/groups → Rule 104
Rule 102: Uses second person 'you do this'?
If Yes: → Website - How-to
If No: → Website - Online Help
Rule 103: Web-native features present (URLs, hyperlinks)?
If Yes: → Website - Catalogue
If No: → Marketing - Product Description
Rule 104: Product categories or general brochure?
Categories → Website - Index
Brochure → Notice - Brochure (General info about organization/product/service)

News Classification (Rules 300-399)

Rule 300: Presenting facts objectively?
If Yes: → Rule 301
If No: → Rule 304 (Author's opinions/emotions take precedence)
Rule 301: Describing a tragedy?
If Yes: → News - Hard News (Natural disaster, crime, etc.)
If No: → Rule 302
Rule 302: Contemporary events?
If Yes: → Rule 303
If No: → News - Feature Article
Rule 303: Light/feature content?
If Yes: → News - Feature Article (Entertaining, light-hearted, human-interest OR in-depth topic coverage)
If No: → News - Press Report
Rule 304: Primarily designed to give the author's point of view?
If Yes: → News - Opinion Piece
If No: → News - Local Interest (Can include author's opinion but primary purpose is to inform locals)

Marketing Classification (Rules 400-499)

Rule 400: Interview format?
If Yes: → Marketing - Interview
If No: → Marketing - Product Description

Review Classification (Rules 500-599)

Rule 500: What's being reviewed?
TV/film/book/music/game → Review - Media Review
Product → Review - Product Review
Service → Review - Service Review
Experience → Review - Experience Review

Legal Classification (Rules 600-699)

Rule 600: Q&A format makes up the majority of text?
If Yes: → Rule 601
If No: → Rule 603 (Could contain some questions/answers but not majority)
Rule 601: Questions rhetorical/for marketing?
If Yes: → Rule 602 (Rhetorical purpose, catching reader's attention)
If No: → Legal - FAQ (Genuine questions with reasonable expectation of answer)
Rule 602: Quiz format?
If Yes: → Legal - Quiz
If No: → Legal - PSA
Rule 603: Speech, form, legislation, or other?
Speech → Legal - Proceedings
Form → Legal - Form
Legislation → Legal - Legislation
Other → Rule 604
Rule 604: Providing timely update on ongoing/recent work?
If Yes: → Rule 605
If No: → Rule 608
Rule 605: Explicitly journal or magazine article?
If Yes: → Legal - Journal Article
If No: → Rule 606
Rule 606: Aimed at the public?
If Yes: → Rule 607
If No: → Legal - Report
Rule 607: About project or organisation?
Project → Legal - Press Release
Organisation → Legal - Newsletter
Rule 608: Explaining how to do something or describing system?
If Yes: → Legal - Guide
If No: → Rule 609
Rule 609: Long-term strategic plan with goals?
If Yes: → Legal - Programme
If No: → Legal - Report

Instructions Classification (Rules 700-799)

Rule 700: Recipe format?
If Yes: → Instructions - Recipe
If No: → Notice - Guide

Subtitles Classification (Rules 800-899)

Rule 800: Movie/TV subtitles?
If Yes: → Rule 801
If No: → Rule 802
Rule 801: Professional or non-professional subtitles?
Professional → Subtitles - Visual Entertainment
Non-professional → Fan Subtitles - Visual Entertainment
Rule 802: DIY/How-to video format?
If Yes: → Subtitles - How-to
If No: → Rule 803
Rule 803: Interview format with Q&A?
If Yes: → Subtitles - Interview
If No: → Rule 804
Rule 804: Podcast, radio or storytelling format?
If Yes: → Rule 805
If No: → Rule 806
Rule 805: Video component likely?
If Yes: → Subtitles - Visual Entertainment
If No: → Subtitles - Audio Entertainment
Rule 806: Speech or presentation?
If Yes: → Subtitles - Talk
If No: → Subtitles - Social Media

Literary Classification (Rules 900-999)

Rule 900: Fictional/story-like?
If Yes: → Literary - Fiction
If No: → Literary - Nonfiction

Medical Classification (Rules 1000-1099)

Rule 1000: For medical professionals?
If Yes: → Medical - Review Article
If No: → Medical - Brochure

Encyclopedia Classification (Rules 1100-1199)

Rule 1100: Technical specs + history?
If Yes: → Encyclopedia - Object Biography
If No: → Rule 1101
Rule 1101: Biographical information?
If Yes: → Encyclopedia - Biography
If No: → Encyclopedia - Reference Material

Other Classification (Rules 1200-1299)

Rule 1200: Purpose: educating, informing, or other?
Educating → Rule 1300 (Didactic)
Informing → Rule 1400 (Notice)
Other Purpose → Other - Unclassified

Didactic Classification (Rules 1300-1399)

Rule 1300: Are answers provided?
If Yes: → Rule 1301
If No: → Didactic - Exam
Rule 1301: For study or reference purposes?
Study → Didactic - Study Notes
Reference → Encyclopedia - Reference Material

Notice Classification (Rules 1400-1499)

Rule 1400: Just addresses/contact information?
If Yes: → Notice - Address
If No: → Rule 1401
Rule 1401: Memorial/obituary content?
If Yes: → Rule 1402
If No: → Notice - Brochure
Rule 1402: Formal obituary format?
If Yes: → News - Obituary
If No: → Social Media - Post
